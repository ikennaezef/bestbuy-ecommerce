# BestBuy Web Store

This is an ecommerce store created from scratch with [Nextjs](https://nextjs.org/), [Sanity CMS](https://sanity.io) and [Chakra UI](https://chakra-ui.com/). It utilizes [stripe](https://stripe.com) for checkout and payments.
